Question - 1: Discuss the scope of var, let and const

Answer:

var:var is called functional scoped.if we declare it inside the function we can access it anywhere of this function .if we declare var outside of any function if become a global variable we can access it any where of the code.


let:let is called block scoped.we can't accessed it outside the block where we declare it.it can be re-assinged


const:const scope also called block scope.we can't accessed it outside the block where we declare it.but it can't be re-assinged.
